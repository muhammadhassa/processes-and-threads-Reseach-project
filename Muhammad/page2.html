What is multiprossessing?

A multiprossessing system has more than two processors.CPUs are applied to the system, which aids in increasing the system's processing speed. Each CPU has its own collection of registers as well as main memory. However, since each CPU is independent, it is possible that one CPU would be idle. One processor may be idle, while the other is overburdened with the specific processes. In a caae like this,the process and the resources are shared dynamically among the processoers.

what is multithreading?

multithreading is a program execution that allows a single process to have multiple code segments like threads. It also runs simultaneously within the process's "context." Multi-threaded programs are those that execute two or more threads simultaneously. As a result, it is sometimes referred to as concurrency.
